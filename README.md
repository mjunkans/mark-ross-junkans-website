# Mark Ross Junkans - Author Website

Personal author website for Mark Ross Junkans, featuring books including "The Return of Mystery: Faith in an Age of Machines" and the "3 Verses a Day" devotional series.

## Pages

- **index.html** - Homepage with book showcase, about section, and newsletter signup
- **bulk-orders.html** - Church and group bulk ordering with pricing tiers
- **speaking.html** - Speaking engagements and media request form

## Deployment

This site is deployed on [Vercel](https://vercel.com).

### To Deploy

1. Push this repository to GitHub
2. Connect the GitHub repository to Vercel
3. Vercel will automatically deploy on each push to main

### Custom Domain (Optional)

To add a custom domain:
1. Go to your Vercel project settings
2. Navigate to "Domains"
3. Add your custom domain and follow DNS configuration instructions

## Local Development

Simply open `index.html` in a web browser to view the site locally.
